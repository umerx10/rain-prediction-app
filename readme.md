# Rain Prediction App 🌦️

A simple web app that predicts whether it will rain based on weather conditions using logistic regression.

## 🚀 Features
- Logistic regression model built from scratch
- Trained on real weather data
- Flask API backend
- Clean HTML frontend
- Predicts based on:
  - Temperature
  - Humidity
  - Wind Speed
  - Cloud Cover
  - Pressure
